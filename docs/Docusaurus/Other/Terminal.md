# Terminal

| Windows  | MacOS / Linux  |  說明 |
|---|---|---|
| cd  | cd  |  切換目錄 |
| cd  | pwd  |  取得目前所在的位置 |
| dir  | ls  | 列出目前的檔案列表  |
|  mkdir |  mkdir |  建立新的目錄 |
| /  | touch  |  建立檔案 |
| copy  | cp  |  複製檔案 |
| move  | mv  | 移動檔案  |
|  del | rm  | 刪除檔案  |
|  cls |  clear |  清除畫面上的內容|
